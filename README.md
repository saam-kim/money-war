# 환율 여행사·수출기업 보드게임

중학교 3학년 사회② 경제 단원 「우리나라 경제와 세계화 - 세계화 시대의 국제 거래와 환율」 수업을 위한 교사용 정적 웹앱입니다. 교사가 노트북에서 실행하고 교실 스크린에 띄운 뒤, 모둠별 토의 결과를 화면에 입력하며 진행합니다.

## 실행 방법

1. 프로젝트 폴더에서 `index.html`을 브라우저로 엽니다.
2. 별도의 서버, 로그인, 데이터베이스, Firebase 설정이 필요 없습니다.
3. 진행 상황은 `localStorage`에 임시 저장되어 새로고침 후에도 이어서 진행할 수 있습니다.
4. `전체 초기화`를 누르면 저장된 진행 상황까지 삭제됩니다.

## 파일 구조

```text
/index.html
/css/style.css
/js/app.js
/js/data.js
/README.md
```

## 주요 기능

- 2~8개 모둠 설정 및 모둠 이름 입력
- 역할 카드 자동 배정
- 총 5라운드 환율 상황 진행
- 모둠별 선택 입력 및 선택 수정
- 라운드별 교사용 진행 안내
- 라운드별 모둠 토의 질문
- 20분 수업용 30초, 45초, 60초 토의 타이머
- 선택지별 추천 역할 표시
- 라운드 결과 화면의 발표 추천 모둠 표시
- 라운드 결과 해설 및 결과 이유 표시
- 국제 거래 유형 정리
- 최종 결과 점수판
- 캡처용 최종 결과 보기
- 웹앱 내 인쇄 자료 화면
- 새로고침 대비 임시 저장
- BOOONG 디자인 시스템 기반 UI

## 디자인 시스템

웹앱 UI는 BOOONG 디자인 시스템의 방향을 반영합니다.

- 브랜드 블루 중심의 색상 체계
- 흰 배경과 부드러운 회색 사이드 톤
- 얇은 보더와 12px 카드 반경
- 교사용 안내, 토의 질문, 개념 해설을 좌측 강조선 callout으로 구분
- 본문은 교과서 느낌의 serif 계열, 버튼과 헤더는 UI sans 계열 사용
- 사진이나 장식 이미지를 제거하고 발표 도구에 맞는 깨끗한 화면 구성

- `환율_여행사_수출기업_보드게임_학생용_학습지.docx`: 이전 버전 학생용 학습지

## 수업 활용 방법

- 교사는 웹앱을 빔프로젝터나 대형 화면에 띄웁니다.
- 학생들은 모둠별로 역할 카드를 확인하고 라운드별 선택지를 토의합니다.
- 교사는 토의 타이머를 켜고, 모둠별 선택을 화면에서 클릭합니다.
- 모든 모둠 선택이 끝나면 결과 화면을 보여 주고 해설을 함께 읽습니다.
- 20분 운영에서는 결과 화면의 발표 추천 모둠 1~2곳만 짧게 발표시키고 다음 라운드로 넘어갑니다.
- 최종 결과 화면에서 `캡처용 보기`를 누르면 수업 기록용 화면으로 정리됩니다.
- 상단 `인쇄 자료` 버튼을 누르면 역할 카드, 라운드 요약, 마무리 발문을 웹에서 바로 인쇄할 수 있습니다.

## 게임 진행 순서

1. 시작 화면에서 수업 설명을 확인합니다.
2. 모둠 수를 2~8개 사이에서 설정합니다.
3. 필요하면 모둠 이름을 직접 입력합니다.
4. 역할 카드를 자동 배정합니다.
5. 역할별 유리한 상황과 불리한 상황을 확인합니다.
6. 각 라운드에서 토의 질문을 보고 모둠별 선택을 정합니다.
7. 결과 화면에서 지표 변화, 결과 이유, 경제 개념 해설을 확인합니다.
8. 최종 결과 화면에서 점수판, 역할별 유불리, 국제 거래 유형을 정리합니다.

## 교사용 운영 팁

- 라운드마다 선택 전 기본 45초 토의 시간을 주세요. 시간이 남는 반은 60초로 늘리면 됩니다.
- 선택 이유를 먼저 말하게 한 뒤 교사가 화면에서 선택하면 수업 참여도가 올라갑니다.
- 결과 화면에서는 추천 모둠 1~2곳만 발표시키고 “역할과 환율 상황이 어떻게 연결되는가?”를 질문하세요.
- 1라운드 환율 상승과 2라운드 환율 하락을 비교하면 원화 가치와 외화 가치 개념을 정리하기 좋습니다.
- 5라운드에서는 문화 콘텐츠 수출이 서비스 거래라는 점을 강조하세요.

## 20분 운영 권장안

- 도입 및 역할 확인: 3분
- 5라운드 진행: 라운드당 약 3분
- 최종 결과 정리: 2분
- 지연 상황 포함 예상 시간: 20~25분

라운드별로 모든 모둠 발표를 받으면 시간이 30분 이상으로 늘어날 수 있습니다. 빠듯하지 않은 20분 운영을 위해 결과 화면의 발표 추천 모둠만 짧게 다루는 방식을 권장합니다.

## 교과 개념 연결

- 국제 거래: 국가 간에 재화, 서비스, 자본, 노동 등 여러 가지 상품을 사고파는 활동입니다.
- 환율: 두 나라 사이의 화폐가 교환되는 비율입니다.
- 환율 상승: 외국 화폐의 가격이 오르고 원화 가치는 하락합니다.
- 환율 하락: 외국 화폐의 가격이 낮아지고 원화 가치는 상승합니다.
- 재화 거래: 자동차, 스마트폰, 식품, 원자재 등 물건을 수출입하는 거래입니다.
- 서비스 거래: 공연, 관광, 운송, 교육 서비스 등의 거래입니다.
- 자본 거래: 해외 주식 투자, 해외 공장 설립 등 자본 이동과 관련됩니다.
- 노동 거래: 외국인 선수, 해외 노동자 등 노동력 이동과 관련됩니다.

## 향후 확장 아이디어

- 교사가 직접 선택지 효과를 조정하는 편집 모드
- 역할 카드와 라운드 카드를 인쇄하는 화면
- 라운드별 발표 모둠 랜덤 선정
- 수업 후 성찰 문장 자동 생성
- 선택 결과를 CSV로 내보내는 기능

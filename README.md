# 환율 배틀: 머니워

경제 단원 「세계화 시대의 국제 거래와 환율」 수업용 교사 웹앱입니다.  
교사가 교실 화면에 띄워 모둠별 토의·선택을 진행하고, 결과와 개념 해설을 함께 확인합니다.

## 바로 실행

→ **[GitHub Pages 링크](https://saam-kim.github.io/currency-fluctuation/)**

별도 서버·로그인·설치 없이 브라우저에서 바로 사용합니다.  
진행 상황은 `localStorage`에 저장되어 새로고침 후에도 이어서 진행할 수 있습니다.

## 파일 구조

```
index.html
css/style.css
js/app.js
js/data.js
```

## 주요 기능

- 2~8개 모둠 설정 및 이름 입력
- 역할 카드 자동 배정 (8종 역할)
- 최대 10라운드 뉴스 기반 환율 상황 진행
- 난이도별 뉴스 구성 (하·중·상)
- 라운드별 모둠 선택 입력 (환율 예측 / 영향 판단 / 전략 선택)
- 모둠 선택 시 화면 깜빡임 없는 즉시 반영
- 30·45·60초 토의 타이머
- 결과 화면: 선택 피드백, 점수 현황, 역할별 유불리 해설
- 최종 화면: 순위, 시상, 판서 정리 도식 (환율 메커니즘 플로우)
- 역할별 유불리 정리 카드
- 수업 마무리 모달 (개념 복습)
- 캡처·인쇄 모드
- 새로고침 대비 상태 자동 저장

## 수업 흐름

1. 시작 화면에서 활동 설명 확인
2. 모둠 수 설정 → 역할 자동 배정
3. 각 라운드: 뉴스 공개 → 45초 토의 → 모둠별 선택 입력 → 결과 해설
4. 최종 화면: 순위 확인 + 판서 정리로 개념 마무리

**20분 운영 기준**: 도입 3분 + 라운드당 3분 × 5라운드 + 마무리 2분
